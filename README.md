# RemoteWeb

## Local Server Installation

Before installing a local server, ensure that Docker is installed in your local machine.
Then, open a terminal and run
```bash
./server_install.sh
```

## Client Installation

Install `remoteweb` package through
```bash
pip3 install -e .
```
or
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
```
